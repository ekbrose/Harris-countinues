⚙ Setup Guide

This tool scrapes publicly available Harris County records. Each source below is free and legally accessible. Follow the setup steps to maximize data coverage.

1
Pre-Foreclosure — Harris County Clerk
The FRCL filings you've been working with come from cclerk.hctx.net. This tool scrapes that public search daily for new FRCL filings by date. No login required. New notices are typically filed 30–45 days before sale dates.
2
Tax Delinquent — hctax.net + HCAD
Tax sale listings are publicly posted at hctax.net before each monthly sale. HCAD bulk property data (owner names, values, account status) downloads free from pdata.hcad.org — updated regularly. This tool cross-references both to build delinquent leads with owner contact info.
3
Probate — Harris County District Clerk
Probate filings are searchable at hcdistrictclerk.com. This tool queries by case type PROBATE daily and extracts new estate filings with property addresses. Probate filings are public record under Texas Government Code §552.
4
Run as a Daily Cron Job (Optional)
To run fully automated, save this file and the companion scraper.py to any computer or server. Set a daily cron: 0 6 * * * python3 /path/to/scraper.py

The Python backend handles HTTP requests, parses HTML, deduplicates, applies your filters, and drops a fresh CSV to your output folder each morning. No subscription needed.
5
Skip Tracing Integration (Optional)
Export your filtered leads CSV directly into BatchSkipTracing or PropStream for phone/email append. The CSV columns are already pre-formatted for both platforms. Typical cost: $0.07–0.18 per record for skip tracing.
